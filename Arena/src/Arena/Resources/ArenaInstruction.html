  • PHP - bot example written in PHP
General information about Arena.
  • Arena is an environment designed for playing games between multiple players. Arena uses settings information from ArenaConfiguration.xml and based on this data runs proper game which is played according to selected elimination rules.
  • Bot is a web application which plays the game. It responds to two calls:
    • GET <bot_url>/info - returns base information about bot and his owner.
      • <string> Name - your bot name
      • <string> AvatarUrl - path to your avatar
      • <string> GameType - type of game that your bot is able to play
      • <string> Description
    • POST <bot_url>/PerformNextMove - bot receives current battlefield/game status (map info, opponents locations, obstacles, etc.) and returns information about its next move.
  • Elimination Plugin - plugin that determines participants and order of games during tournament. Currently there are two elimination plugins implemented:
    • TournamentLadder - players are grouped in pairs and play duels. The winner from each pair goes to the next stage, loser leaves in sorrow
    • RoundRobin - each player plays specified number of duels with every other player. Players are evaluated by their number of wins and loses
  • Game Plugin - plugin that determines the type of game which bots are playing
  • ArenaConfiguration.xml- XML configuration file used by Arena, contains Bots links and some settings for Game and Elimination Controls.
    • EliminationConfiguration
      • Type - runs selected elimination.
    • GameConfiguration
      • Type - runs selected game.
      • NextMoveDelay - time in miliseconds between executing bot moves. Lower value means faster gameplay
    • ArenaMessageDuration - duration of the arena messages, e.g duel result
    • BotUrls - bot addresses ex.
      • <url>http://localhost:53886/TicTacToeBot/</url>
      • <url>http://67221-tpe540:81/tankblasterbotveronica/</url>
Bot written in .Net
  • Setup sample bot on IIS server
    1. Open some bot project from "dotNet" folder, rebuild it or restore packages for solution and then rebuild. Run solution. Bot should be visible in IIS server. This is the moment you can test your bot localy.
    2. Open IIS and create new website using binaries from "dotNet" folder project - right click on "Sites" -> "Add Web Site"
    3. Change Port number to some unused port (eg. 82)
    4. Type in website name
    5. Set physical path to your bot application folder
    6. Click OK.
    7. Right click on your bot web site -> "Edit Permissions" -> "Security" bookmark
    8. Click "Edit" and then "Add" to grant permission to user. Dialog will appear, type "IIS APPPOOL\{your application name}" in text box. Change Location for your local host location ie. computer name. Click Ok.
    9. Choose your user and grant him Full control, click OK and go back to IIS.
    10. Run sample bot (if not already running)
    11. Make sure that Windows Features (Accessible from Add/Remove programs in Control Panel) -> Internet Information Services -> WWW services -> Asp.Net 4.5 is checked.
    12. Check your bot in web browser eg. "http://localhost:82/info"
  • Modify ArenaConfiguration file:
    1. change game type you wish to play
    2. change elimination type name to your preferred elimination type
    3. modify bot urls. If you need more players, just type a few copies of the same url. They will be interpreted as a separate bots
  • When you run the arena, it should automatically connect with bots. Press "Play" to start game
Bot written in PHP
  • Setup sample bot on Apache server
    1. Download WAMP server from http://www.wampserver.com/ and install it
    2. Copy folder "TankBlasterBot" (hello_wars/PHP/TankBlaster/) to your WAMP server www folder (default "C:/wamp/www" on Windows)
    3. Run WAMP
    4. Your bot should be accessible at "http://localhost/tankblasterbot/". Check "http://localhost/tankblasterbot/info" in your browser to see if it works
  • Modify ArenaConfiguration file:
    1. change game type you wish to play
    2. change elimination type name to your preferred elimination type
    3. modify bot urls. If you need more players, just type a few copies of the same url. They will be interpreted as a separate bots
  • When you run the arena, it should automatically connect with bots. Press "Play" to start game
Bot written in JAVA
  1. Import bot from HELLOWARS/JAVA folder to your workspace
  2. Find InetSocketAddress method in TankBlasterBot class and change port number to some valid port number

  4. You can check if bot is running typing http://localhost:9065/info in webbrowser

Bot written in Node.js
  • Setup sample bot in Node.js
    1. Download Node.js from https://nodejs.org/en/ and install it
    2. Copy folder "TankBlasterBot" (hello_wars/nodeJs/TankBlasterBot/) to your local directory
    3. Open command prompt and go to project directory
    4. Type "npm install" to install all necessary modules
    5. Type "node app.js" to run application
    6. Your bot should be accessible at "http://localhost:8081/". Check "http://localhost:8081/info" in your browser to see if it works
  • Modify ArenaConfiguration file:
    1. change game type you wish to play
    2. change elimination type name to your preferred elimination type
    3. modify bot urls. If you need more players, just type a few copies of the same url. They will be interpreted as a separate bots
  • When you run the arena, it should automatically connect with bots. Press "Play" to start game
Firewall configuration
  • If you want your bot to be visible from your network, configure your firewall:
    • open Windows Firewall -> Advanced Settings -> Inbound Rules - add exception for port on which your bot runs
    • configure other firewall programs if installed.
